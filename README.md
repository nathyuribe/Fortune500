# Fortune500
Structured dataset of 100–500 companies including: company name, HQ country, industry, the most recent three fiscal years, revenue and revenue units, plus 3–5 optional KPIs reported in the same format. All values must match the original source data, preserving reported years, units, and currencies.

# Data set

https://www.kaggle.com/datasets/rm1000/fortune-500-companies/data
Accessed with kaggle library and API 

- The initial dataset is included in the folder for easy access
- The resulting dataset is included as well for the same reason

# Notes
This dataset didn't have countries available, but had cities, all within the US so that's what I used for the excersise
